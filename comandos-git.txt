1 - git init: para iniciar repositório local
2 - git config user.name: criar usuário local
    git config --global user.name: criar usuário global
3 - git config user.email: criar email do usário local
    git config iiglobal user.email: criar email do usário global
4 - git add --all ou -A: adicionar arquivos ao commit
5 - git commit -m "a mensagem que identifica o commit"
6 - git branch: saber quantas branchs tem
7 - git branch <nome da branch> : criar uma nova branch
8 - git branch -m <branch atual> <novo nome>: renomear branch
9 - git checkout <nome da branch> : mudar de branch
10 - git merge <nome da branh> : juntar as alteraçoes feitas numa branch 
11 - git config --global --replace-all user.name: alterar nome de usuário global
